# Netflix-GPT

A sleek, Netflix clone built purely with frontend technologies. This project uses React, Redux, Tailwind CSS, TMDB for video data, and Firebase for hosting/authentication. I will update and add backend also later..

## Features


- **State Management:** Powered by Redux for smooth state transitions.
- **Video Data:** Fetches movie/TV show data from The Movie Database (TMDB) API.
- **Firebase Integration:** Uses Firebase for authentication and hosting.
- **Frontend Only:** No Node.js or custom backend involved – purely frontend magic.

---

## Tech Stack

- **React** – UI library for building interactive interfaces.
- **Redux** – State management library.
- **Tailwind CSS** – Utility-first CSS framework.
- **TMDB API** – Provides movie and TV show data.
- **Firebase** – Authentication and hosting.
